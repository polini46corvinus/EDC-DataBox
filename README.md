# EDC DataBox
Ez a célszoftver adatelőkészítési és adategyesítési funkciókat lát el.

## Előkövetelmények:
* Python 3.14.2 (vagy újabb) — https://www.python.org/downloads/windows/
  * Tkinter (része a Python-csomagnak)
* Numpy 2.4.0 (vagy újabb)
* Pandas 2.3.3 (vagy újabb)
* Customtkinter 5.2.2 (vagy újabb)
